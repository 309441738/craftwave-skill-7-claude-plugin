# 奢华邮轮体验推荐 · Claude Code 插件

专为追求奢华的旅行者推荐高端邮轮体验，提供顶级设施与服务。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/craftwave-skill-7-claude-plugin
/plugin install craftwave-skill-7@craftwave-skill-7-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **奢华邮轮体验推荐**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
